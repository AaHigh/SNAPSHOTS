### Patents to Document

1. **Arcade Game with Keypad Input**
   - Patent Numbers: US6287201B1, US6572477B2, US20010055991A1, WO2000053280A8, EP1159045A1
   - Priority Date: 1999-03-12
   - Original Assignee: Atari Games Corp / Midway Games West Inc
   - Current Assignee: Warner Bros Entertainment Inc
   - **Core Innovation:** Numeric keypad-based player data persistence system. Players enter unique codes to retrieve stored preferences (vehicle type, track selection, control scheme) and game statistics. Database updates after each play session, creating a linked arcade cabinet ecosystem.
   - **Value Proposition:** Early implementation of player profile persistence in arcade systems; prototype for modern player account/identity systems.

2. **Ghost or Phantom Vehicle Racing Mechanic**
   - Mentioned in patent search results related to arcade racing games
   - **Core Innovation:** Previous vehicle race sequences stored and played back as ghost/phantom vehicles simultaneously with current player vehicle. Reward level dynamically adjusted based on player ability.
   - **Value Proposition:** Foundational mechanic for competitive racing games; enables asynchronous multiplayer and skill-based difficulty balancing.

3. **Wearable Display System and Method**
   - Patent Number: US9707480B2
   - Publication Date: 2016-05-13
   - Assignee: IGT (International Game Technology)
   - **Core Innovation:** Wearable display device with vision direction data processing. Display buffer receives data from graphics engine regarding wearer position in virtual environment. Display processor unit handles vision direction using graphics engine data.
   - **Value Proposition:** Head-mounted display system for immersive gaming; applicability to VR/AR environments; integration of positional awareness with display rendering.
